1. Al día 23/02/2026 Nos sentamos en grupo y concretamos entre todos que nos repartiriamos de manera equitativa los RF,

Ana Isabella Garcia Rozo se le asigna: RF01 – Filtro avanzado de tareas
El sistema deberá permitir:
• Filtrar tareas por estado (pendiente, en proceso, completada).
• Filtrar tareas por usuario.
• Combinar filtros simultáneamente.
El filtro debe actualizar la interfaz sin recargar la página.

Andrés Santiago Calvete Lesmes se le asigna: RF02 – Ordenamiento dinámico
El sistema deberá permitir ordenar las tareas por:
• Fecha de creación.
• Nombre de la tarea.
• Estado.

y también RF03 – Sistema de notificaciones estructurado
Se deberá implementar un módulo independiente de notificaciones que:
• Muestre mensajes de éxito, error e información.
• Permita reutilización en otros módulos.
• No dependa directamente del módulo API.

Fernando Andrés Rodriguez Salamanca se le asigna: RF04 – Exportación de tareas
El sistema deberá permitir exportar la lista de tareas visibles en pantalla en formato JSON
descargable.
La funcionalidad deberá respetar la separación de responsabilidades y no mezclar lógica
de interfaz con lógica de procesamiento de datos.
